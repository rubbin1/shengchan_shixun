# 基于颜色与形状的零件检测分离系统

## 概述

本项目实现了一个基于 OpenCV 的工业零件检测系统，能够从图像中识别不同颜色的零件，分离粘连零件，并标注其形状类型。

## 运用到的算法与方法

### 1. 颜色空间转换与阈值分割（HSV 颜色掩膜）

将图像从 BGR 色彩空间转换到 HSV 色彩空间，利用 HSV 各通道设定阈值范围生成二值掩膜，实现对特定颜色零件的提取。

- **为什么用 HSV**：相比 BGR/RGB，HSV 对光照变化更鲁棒，颜色阈值设定更直观。

### 2. 形态学操作（开运算 & 闭运算）

对二值掩膜先后进行闭运算和开运算：

- **闭运算**（先膨胀后腐蚀）：填充目标内部的小孔洞和缝隙。
- **开运算**（先腐蚀后膨胀）：消除小噪点，断开细窄连接。

两个操作组合使用，在保留目标轮廓的前提下清理掩膜中的噪声。

### 3. 自适应腐蚀 + 分水岭分离（`erosion_watershed_separation`）

用于处理零件之间的**边-边贴合**（凸缺陷方法难以检测的粘连类型）：

1. 逐步加深腐蚀（1~3 次迭代），断开零件间的薄连接。
2. 通过连通域分析检测是否成功分离为多个独立区域。
3. 借助**碎片化检测**（最小组件面积 < 最大组件的 15%）判断是否腐蚀过度。
4. 以腐蚀结果的连通域作为种子标记（markers），执行 **Watershed（分水岭）算法**，将原始掩膜精确分割为独立零件。

### 4. 凸缺陷检测与轮廓切割（`split_contour_by_convexity_multi`）

用于处理具有明显凹陷特征的粘连轮廓：

1. **轮廓平滑**（`approxPolyDP`）：降低轮廓噪声对凸缺陷检测的干扰。
2. **凸包**（Convex Hull）与**凸缺陷**（Convexity Defects）：计算轮廓的凸包，检测凹陷位置和深度。
3. 筛选深度超过阈值（`SEPARATION_STRENGTH × min_side`）的凹点。
4. 对凹点进行**空间聚类**，合并距离过近的点（避免在同一个凹陷处生成多个切割点）。
5. 沿凹点切割轮廓，将一个大轮廓分割为多个子轮廓。
6. 过滤无效子轮廓（面积过小或周长过短）。

### 5. 距离变换 + 分水岭（`improved_watershed`）

备用分离方案（当前代码中未直接调用，作为工具函数保留）：

1. 对二值掩膜进行**距离变换**（Distance Transform），计算每个前景像素到最近边界的距离。
2. 使用高阈值（`DIST_THRESH_RATIO × max_distance`）提取各零件的中心区域作为"确定前景"（sure foreground）。
3. 通过连通域过滤和轻度腐蚀，确保种子标记不粘连。
4. 执行分水岭算法完成分割。

### 6. 轮廓多边形逼近（`fit_polygon_exact`）

通过二分查找 epsilon 参数，使用 `approxPolyDP` 将轮廓多边形拟合到指定的目标顶点数（三角形=3，四边形=4，六边形=6）。

### 7. 两级分离策略（`separate_touching_parts`）

整合上述分离方法的决策流程：

1. **第一级**：调用腐蚀+分水岭（`erosion_watershed_separation`）断开薄连接。
2. **第二级**：对剩余的大面积轮廓，评估其 **solidity**（轮廓面积/凸包面积比值）和深凹点数量：
   - `solidity > 0.95`：几乎完美凸形 → 不拆分
   - `solidity < 0.88`：明确凹形 → 执行凸缺陷切割
   - `0.88 ≤ solidity ≤ 0.95` 且有 2~4 个深凹点：边-边贴合 → 切割
   - `solidity > 0.92` 且有 5 个以上凹点：轮廓噪声 → 不拆分
3. 切割结果验证：子轮廓面积比不能过于悬殊（最小/最大 ≥ 0.2），否则保留原轮廓。

## 主程序流程

```
读取图像 → 缩放显示 → BGR→HSV 转换
    ↓
对每种颜色（绿/粉/浅橙/橙）：
    ├─ 1. 颜色掩膜提取（cv.inRange）
    ├─ 2. 形态学清理（闭运算 + 开运算）
    ├─ 3. 多策略粘连分离（separate_touching_parts）
    │      ├─ 腐蚀+分水岭（断开薄连接）
    │      └─ 凸缺陷切割（分割凹形粘连）
    └─ 4. 结果标注
           ├─ 计算每个轮廓的质心
           └─ 标注颜色名 + 形状名
    ↓
显示结果图像
```

### 颜色-形状对应关系

| 颜色 | 目标形状 | 边数 |
|------|---------|------|
| 橙色 | 六边形 | 6 |
| 浅橙色 | 三角形 | 3 |
| 绿色 | 四边形 | 4 |
| 粉色 | 四边形 | 4 |

## 可调参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `KERNEL_OPEN` | (3, 3) | 开运算核大小 |
| `KERNEL_CLOSE` | (1, 1) | 闭运算核大小 |
| `MIN_AREA_ABS` | 500 | 最小面积绝对阈值（相对 800×600） |
| `SEPARATION_STRENGTH` | 0.03 | 凸缺陷深度系数，越小越敏感 |
| `DIST_THRESH_RATIO` | 0.12 | 距离变换前景阈值比例 |
| `MIN_MARKER_AREA` | 20 | 分水岭标记最小面积 |
| `DEBUG_MODE` | True | 是否显示中间调试图像 |

## 依赖

- Python 3.x
- OpenCV (`opencv-python`)
- NumPy
